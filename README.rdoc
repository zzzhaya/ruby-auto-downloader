= image-crawler

* http://github.com/zzzhaya/ruby-auto-downloader/

== DESCRIPTION:

Crawl sites and download images/pictures or everything you want

== FEATURES/PROBLEMS:

* FC2 blog crawling

== SYNOPSIS:

bin/fc2 --path ~/images --min 20 http://xxx.blog00.fc2.com/blog-entry-xxxx.html

== REQUIREMENTS:

* nothing so far

== INSTALL:

* sudo gem install auto-downloader
